# Tasman

Tasman is a comprehensive project that aims to resolve the problem that people nowadays tend to overload too many tasks. It is also designed specifically to avoid burnouts, especially for workers, university students. It is an efficient and helpful piece of software that is able to create new tasks, set the due date for the task, check the current status of specific tasks, and  many more features when diving into the web app. This project is divided into two main components: Backend and Frontend.

## Features
- **Backend**: Built with NodeJS + ExpressJS for dynamic data manipulation and RESTful APIs. 
- **Frontend**: User-friendly ReactJS interface powered by Vite. 
- **Database**: PostgreSQL for reliable, real-time data storage and queries. 
- **Authentication**: Secure login and registration using JWT. 
- **Deployment**: Frontend hosted on Vercel, backend on Render.

## Tech Stack 
- Frontend: React, Vite, TypeScript 
- Backend: NodeJS, ExpressJS 
- Database: PostgreSQL 
- Authentication: JWT 
- Deployment: Vercel (frontend), Render (backend)
