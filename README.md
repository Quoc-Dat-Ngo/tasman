# Tasman

Tasman is a comprehensive project aims to resolve the problem that people nowadays tend to overload too many tasks. It is also designed specifically to avoid burnouts, especially for wokers, university students. It is an efficient and helpful piece of software that is able to create new task, set the due date for the task, check the current status of specific task,etc... many more features when diving into the web app. This project is divied into two main components: Backend, Frontend.

## Features
* Backend: Dynamic manipulation and presentation of the data through a NodeJS + ExpressJS application.
* Frontend: Seamless integration with a user-friendly ReactJS interface.
* Database: Real-time data manipulation within a Postgres database using SQL queries.
